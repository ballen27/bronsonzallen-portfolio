The Tessera

A web based interactive game that helps teach young students about math, science, and problem solving.

The Tessera is a colaboration of artists, writers, deisgners, and developers across the US to design an interactive web game for young students. As a developer on the project I implemented the ideas of artists and game designers into javascript using AngularJS. Tis includes paralax scrolling of images, and puzzle game mechanics all built form scratch from the designers minds onto real webpages .

The Tessera game is available at www.thetessera.org.
